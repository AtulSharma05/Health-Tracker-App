# NutriPal Frontend

Flutter app for NutriPal nutrition tracking.

## Features
- Login and registration with backend auth
- Meal logging with MongoDB persistence
- Meal history from backend
- Nutrition plan generation via backend endpoint
- Nutrition dashboard and analytics UI

## Run
1. Start backend first (project2/backend)
2. cd frontend
3. flutter pub get
4. flutter run

## API URL
- Android emulator default uses http://10.0.2.2:4000/api/v1
- Web default uses http://localhost:4000/api/v1
